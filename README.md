### add
    #### Description

    Write a function that returns the sum of two numbers.

    #### Example:

    ```Javascript
    param1 = 1
    param2 = 2
    add(param1, param2) = 3.
    ```

    ## Input/Output

    - **[time limit] 3000ms (java)**
    - **[input] integer param1** <br>
    *Guaranteed constraints:* <br>
    ```Javascript
    -100 ≤ param1 ≤ 1000
    ```
    - **[input] integer param2** <br>
    *Guaranteed constraints:* <br>
    ```Javascript
    -100 ≤ param2 ≤ 1000
    ```
    - **[output] integer** <br>
    The sum of the two inputs.
